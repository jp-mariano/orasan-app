# Contributing to Orasan

Thanks for helping Orasan! We welcome PRs for core features and `/ee` (premium).

## Guidelines

- Fork and clone the repo.
- Use `pnpm` for dependencies.
- Follow TypeScript/ESLint rules.
- PRs to `/ee` don’t grant premium access (see `/ee/LICENSE`).

## Issues

File bugs or ideas at [Issues](https://github.com/jp-mariano/orasan-app/issues).
