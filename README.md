# Orasan

Privacy-first, open-source X (Twitter) reminders and time tools.

## Features

- **X-Reminders**: DM `@OrasanApp` (e.g., "remind me in 30m").
- **Time Tools**: Timer, stopwatch, world clock, local time.
- **Open-Source**: Core features under MIT, premium in `/ee` (proprietary).

## Getting Started

1. Clone: `git clone https://github.com/jp-mariano/orasan-app`
2. Install: `pnpm install`
3. Run: `pnpm build`
4. Run: `pnpm dev`
